# 🐘 Hasthi.lk — Elephant Orphanage Management System

A full-stack web application for managing an elephant orphanage — covering elephant registration, adoption, donations, event management, health tracking, and visitor entry bookings.

---

## 🛠️ Tech Stack

### Frontend
- **React 19** (Vite)
- **Tailwind CSS**
- **Framer Motion** — animations
- **Recharts** — analytics charts
- **FullCalendar** — event calendar
- **Lucide React** — icons
- **React Router DOM** — routing
- **Axios** — API calls
- **jsPDF** — PDF generation

### Backend
- **Node.js + Express 5**
- **MongoDB + Mongoose**
- **JWT** — authentication
- **Stripe** — payment processing
- **Cloudinary** — image hosting
- **Nodemailer** — email notifications
- **PDFKit** — certificate generation
- **QRCode** — QR code generation
- **Google Generative AI** — AI chat
- **Google Auth Library** — Google login

---

---

## 🚀 Features

| Module | Description |
|---|---|
| 🔐 **Authentication** | JWT-based login/register with role-based access (Admin, Manager, Caretaker, Vet, Event Manager, User) |
| 🐘 **Elephant Management** | Register, view, and manage elephants with health records |
| 💚 **Adoption** | Users can request to adopt elephants; admin approves/rejects |
| 💰 **Donations** | Stripe-powered donation system with receipts |
| 🎫 **Entry Bookings** | Visitors can book entry tickets with online payment |
| 📅 **Event Management** | Create, manage, and book events with calendar view |
| 🩺 **Health Requests** | Caretakers report issues; vets and managers review |
| 📊 **Analytics** | Dashboards with charts for admin, manager, and event manager |
| 📧 **Email Notifications** | Automated emails via Nodemailer |
| 🤖 **AI Chat** | Google Generative AI powered chat assistant |
| 📍 **Google Maps** | Location integration for the orphanage |
| 📄 **PDF Reports** | Generate adoption certificates and reports |
